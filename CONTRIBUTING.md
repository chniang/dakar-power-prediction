# Guide de contribution

Merci de votre intérêt pour ce projet !

## Comment contribuer

### Signaler un bug

1. Vérifiez que le bug n'a pas déjà été signalé dans les Issues
2. Créez une nouvelle Issue avec le template "Bug"
3. Décrivez le problème en détail avec :
   - Les étapes pour reproduire
   - Le comportement attendu
   - Le comportement observé
   - Captures d'écran si pertinent

### Proposer une amélioration

1. Créez une Issue avec le template "Feature Request"
2. Décrivez clairement la fonctionnalité souhaitée
3. Expliquez pourquoi elle serait utile

### Soumettre du code

1. Forkez le projet
2. Créez une branche (`git checkout -b feature/AmazingFeature`)
3. Committez vos changements (`git commit -m 'Add AmazingFeature'`)
4. Pushez vers la branche (`git push origin feature/AmazingFeature`)
5. Ouvrez une Pull Request

## Standards de code

- Utilisez Python 3.12+
- Suivez PEP 8
- Documentez les fonctions avec docstrings
- Testez votre code avant de soumettre
